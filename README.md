# Weather-Recording-Automation-System
## 气象记录自动化系统
>这是一个自动将传感器记录的数据分析汇总，最终以数据可视化的方式显示出数据。通过时间的记录，越能展示当地的气象特征，大数据分析将越来越精准。

> 是一个比较有意思的小项目，方案不算复杂，设计也存在一些值得改进的地方。
